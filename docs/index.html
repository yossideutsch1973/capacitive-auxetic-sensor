How It Works

Auxetic Lattice Mechanics

Auxetic materials exhibit negative Poisson's ratio, expanding laterally when stretched longitudinally. Our re-entrant honeycomb structure achieves this through carefully designed geometric parameters.

Key Parameters

  • Re-entrance Angle (Œ±): Controls auxetic behavior strength
  • Cell Size (a, b): Determines structural scale and sensitivity
  • Wall Thickness: Affects mechanical robustness
  • Array Configuration: Scales sensor size and output

Capacitive Sensing

Deformation changes the gap between capacitor plates, providing a measurable electrical signal proportional to applied mechanical load.
